# Lockdown

{% hint style="danger" %}
The command must be used in the channel you want to be locked or unlocked
{% endhint %}

## Lock

### Usage

```
b!lock {reason}
```

### **Permissions**

```
Manage Channel
```

### **Arguments**

```
{reason} - The reason for the channel to be locked
```

## Unlock

### Usage

```
b!unlock
```

### **Permissions**

```
Manage Channel
```

### **Arguments**

```
None
```

## Syntax

```
[] - Required
{} - Optional
```
