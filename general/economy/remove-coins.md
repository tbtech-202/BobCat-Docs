# Remove Coins

### **Description**

Removes coins to another member

### Usage

```
/economy remove-coins [member] [amount]
```

### Permissions

```
None
```

### Arguments

<pre><code><strong>[member] - A mention or ID
</strong><strong>[amount] - The amount you want to remove</strong></code></pre>

### Syntax

```
[] - Required.
{} - Optional.
```
