# Give Coins

### **Description**

Give coins to another member

### Usage

```
/economy give-coins [member] [amount]
```

### Permissions

```
None
```

### Arguments

<pre><code><strong>[member] - A mention or ID
</strong><strong>[amount] - The amount you want to give</strong></code></pre>

### Syntax

```
[] - Required.
{} - Optional.
```
