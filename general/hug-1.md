# Activity

### **Description**

Start an activity in the current voice channel

### Usage

```
/activity [channel] [activity]
```

### Permissions

```
None
```

### Arguments

<pre><code><strong>[channel] - A voice channel to start the activity in
</strong><strong>[activity] - The activity to start</strong></code></pre>

### Syntax

```
[] - Required.
{} - Optional.
```
