# User

### **Description**

Shows your or a member information and avatar.

### Usage

<pre><code>/user {member}

<strong>App Command
</strong>Right click a user > Apps > User</code></pre>

### Permissions

```
None
```

### Arguments

```
{member} - A member mention or ID
```

### Syntax

```
[] - Required.
{} - Optional.
```
