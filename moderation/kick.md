# Kick

### **Description**

Kicks a member from the server

### Usage

```
b!kick [member] {reason}
```

### Permissions

```
`Kick Members` permissions
```

### Arguments

```
{Mention} - A user mention or User ID.
[Reason] - A reason for kick.
```

### Syntax

```
[] - Required.
{} - Optional.
```
