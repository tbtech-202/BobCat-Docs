# Role removeall

### **Description**

Removes all roles from the specified member.

### Usage

```
b!roll removeall [member] [role]
```

### Permissions

```
`Manage Roles` permissions
```

### Arguments

```
[member] - A member mention or ID
[role] - A role mention or ID
```

### Syntax

```
[] - Required.
{} - Optional.
```
