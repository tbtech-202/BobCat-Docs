# Duplicate Message

### **Description**

This will duplicate a selected message

### Usage

```
App Command
Right click a message > Apps > Duplicate Message
```

```
`Manage Messages` permissions
```

### Arguments

```
[message] - The selected message
```

### Syntax

```
[] - Required.
{} - Optional.
```
