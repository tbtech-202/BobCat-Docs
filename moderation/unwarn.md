# Unwarn

## Usage

```
b!unwarn [member] {case_id}
```

## **Permissions**

```
Moderate Members
```

## **Arguments**

```
[member] - A user mention or User ID
{case_id} - The #<num> is the case id
```

## Syntax

```
[] - Required
{} - Optional
```
