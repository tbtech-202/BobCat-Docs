# Unwarn

### **Description**

unwarns a member

### Usage

```
b!unwarn [member] [case_number]
```

### Permissions

```
`Moderate Members` permissions
```

### Arguments

```
[member] - A member mention or ID
[case_number] - The warn you want to remove
```

### Syntax

```
[] - Required.
{} - Optional.
```
