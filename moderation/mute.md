# Mute

## Usage

```
b!mute [member] {reason}
/mute [member] {reason}
```

## **Permissions**

```
Ban Members 
```

## **Arguments**

```
[member] - A user mention or User ID
{reason} - A reason for the action
```

## Syntax

```
[] - Required
{} - Optional
```
