# Mass Ban

### **Description**

Bans more the one member at a time from the server

### Usage

```
b!massban [members] {reason}
```

### Permissions

```
`Ban Member` permissions
```

### Arguments

```
[members] - User IDs
{reason} - Reason for the mass ban
```

### Syntax

```
[] - Required.
{} - Optional.
```
