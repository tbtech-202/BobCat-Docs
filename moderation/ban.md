# Ban

## Usage

```
b!ban [member] {reason} {delete_message}
/ban [member] {reason} {delete_message}
```

## **Permissions**

```
Ban Members
```

## **Arguments**

```
[member] - A user mention or User ID
{reason} - A reason for the action
{delete_message} - How many days you want to delete the users messages
```

## Syntax

```
[] - Required
{} - Optional
```
